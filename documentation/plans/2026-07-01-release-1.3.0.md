# Release 1.3.0 — version bump, changelog, documentation accuracy

## Objective

Cut the fourth release of Draw: stamp the build `1.3.0`, document the eight features merged since
`1.2.0` in the changelog and roadmap, and keep the public-facing docs in line with reality. No
build/distribution machinery is added here — that remains the separately-tracked public-release blocker.

## Context

`main` carries **16 commits across 8 feature branches** on top of the `1.2.0` tag, all
backward-compatible, user-facing additions/refinements with **no model or serialization change**, so
under Semantic Versioning this is a **MINOR** bump → **1.3.0**:

- `945d4b2` — **`:` command palette** replaces the bottom command bar with a centered overlay; also
  **rebinds the style picker Shift+Y → Shift+T**.
- `382a7fd` — **recent documents** listed on the empty-state welcome screen (click to open, hover × to
  remove).
- `73668a9` — **selection palettes open with no selection** (Shift+I / Shift+T / Shift+A open on an
  active document; inapplicable controls disabled).
- `5434f7b` — **Shift+A align & distribute palette** (+ Shift+H help refresh).
- `e533a06` — **duplicate split**: `Ctrl+D` without connectors, `Ctrl+Shift+D` with connectors.
- `84d0ae9` — **automatic connector spacing** on connect / duplicate / paste / mind-map "+"
  (`EditorOptions.AutoSpaceConnectors`, default on).
- `6c958b2` — **mind-map branch dash styles** (Dash / Dot / DashDot rendered on tapered ribbons).
- `f682b84` — **empty-state guidance** + the **Shift+H** help overlay works with no document open.

## Decisions

- **Version:** `1.3.0` in `Directory.Build.props` (single source of truth, solution-wide) and a
  matching `1.3.0.0` in `src/Draw.App/app.manifest` (Win32 SxS identity; cosmetic for .NET but kept in
  sync). The in-app label needs no change — `ShortcutHelpViewModel.VersionLabel` reads the assembly.
- **Scope:** documentation + metadata only. No publish script, no CI/CD, no packaging — source + tag
  only, as with `1.0.0` / `1.1.0` / `1.2.0`. (Confirmed with the owner for this release.)
- **Changelog categories:** *Added* — recent-documents welcome, Shift+A align/distribute palette,
  automatic connector spacing, empty-state guidance. *Changed* — `:` command palette (replaces the bar)
  and Shift+Y→Shift+T, the Ctrl+D / Ctrl+Shift+D duplicate split, and palettes opening with no
  selection (all are changes to existing behaviour/UX). *Fixed* — mind-map branch dash styles (the dash
  style was previously ignored on branch ribbons).
- **Roadmap:** the auto-space and command-palette sections lost their "pending merge" caveats (now
  merged); the stale *Duplicate carries either-end connectors* paragraph under *Copy/paste + image
  support* was corrected to the new `Ctrl+D` / `Ctrl+Shift+D` split; new sections were added for the
  five features that had none; and a new `Release 1.3.0` section summarises the cut.
- **Visual verification:** all eight features are GUI-only. WSL2 cannot render the app, so the Shift+H
  version check below is the owner-side manual step; the headless build + pure-logic test suite are the
  automated gates.
- **Conventions:** follows the project scheme (`documentation/roadmap.md` + dated
  `documentation/plans/*.md`), mirroring `2026-06-30-release-1.2.0.md`. Tags carry no `v` prefix.
