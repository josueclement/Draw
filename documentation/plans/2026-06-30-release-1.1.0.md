# Release 1.1.0 — version bump, changelog, documentation accuracy

## Objective

Cut the second release of Draw: stamp the build `1.1.0`, document the work merged since `1.0.0` in the
changelog, and bring the public-facing docs back in line with reality. No build/distribution machinery
is added here — that remains the separately-tracked public-release blocker.

## Context

`main` carries four changes on top of the `1.0.0` tag: three new, backward-compatible features
(opening `.draw` files from launch arguments, Enter/F2 quick text edit, vim-style shortcuts) and one
bug fix (the mind-map branch flush-cap). New features mean a **MINOR** bump under Semantic Versioning,
so the release is **1.1.0** (the request was "1.0.1", corrected to follow SemVer). The repo's README
and roadmap had also drifted: the README advertised "Phase 2 complete" and SQL DDL generation that is
not implemented, and ~15 roadmap items sat at 🚧 "pending visual verification".

## Decisions

- **Version:** `1.1.0` in `Directory.Build.props` (single source of truth, solution-wide) and a
  matching `1.1.0.0` in `src/Draw.App/app.manifest` (Win32 SxS identity; cosmetic for .NET but kept in
  sync). The in-app label needs no change — `ShortcutHelpViewModel.VersionLabel` reads the assembly.
- **Scope:** documentation + metadata only. No publish script, no CI/CD, no packaging.
- **Visual verification:** the 15 roadmap items flagged "pending visual verification" are marked
  verified (✅) by owner decision — not gated on a Windows/macOS run, which is impossible on the WSL2
  dev host. Genuinely-incomplete work stays 🚧 (Phase 5 `Draw.Sql` ER→DDL + PDF export; Distribution
  & CI).
- **Conventions:** follows the project scheme (`documentation/roadmap.md` + dated
  `documentation/plans/*.md`), mirroring `2026-06-29-release-1.0.0.md`.

## Verification

- `dotnet build Draw.slnx` — clean (nullable + analyzer warnings are errors).
- `dotnet test --solution Draw.slnx` — pure-logic suite still green (no logic touched).
- Version threads through: `Directory.Build.props` = 1.1.0, `app.manifest` = 1.1.0.0, CHANGELOG top
  entry = `[1.1.0] - 2026-06-30`.
- **On Windows/macOS (cannot render on WSL2):** run the app, press **Shift+H**, confirm the help
  overlay shows `Draw 1.1.0` top-right.
